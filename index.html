Application Features

Search Questions

Questions can be searched based on multiple attributes like Subject, Language , Grades, Marks, Tags

One or combination of multiple search attributes help in selecting questions for right term

Download Questions

You can organize questions by reviewing their ratings, rate them, delete them

Questions can be downloaded in pdf format for easy distribution and printing

Submit Questions

Advantage of adding a question is you can organize questions per your need using searchable tags. Other students can benifit from contributions
